# FitnessMate

FitnessMate is a fitness tracking web application designed to help users monitor their fitness journey through meal logging, workout tracking, calorie goals, protein goals, and progress monitoring.

## Project Status

In progress — currently in the planning, documentation, and database design stage.

## Purpose

The purpose of FitnessMate is to support users who want to bulk, maintain weight, or lose weight by helping them track their daily nutrition, workouts, and body progress in one place.

## Planned Features

- User registration and login
- Fitness profile creation
- Goal selection: bulking, maintaining weight, or losing weight
- Daily calorie and protein target tracking
- Manual meal logging
- Workout logging
- Progress photo tracking
- Body measurement tracking
- Dashboard summary for meals, workouts, and progress

## Planned Tech Stack

- Frontend: React
- Backend: Node.js / Express.js
- Database: MySQL
- Version Control: Git and GitHub
